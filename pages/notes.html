Release Notes

Release Notes for FL Studio 20.5

This is a sample release notes document created for portfolio purposes.

Release Overview

This update includes enhancements to existing features, new features and functionality, and minor bug fixes.

What's New?

  • Layout: Users can now customize their layout by adding an image to their background or changing the background color.

  • VST Plugins: Free VST instrument plugins have been added: Autogun, GMS, and Harmor.

  • Shop: Purchasing plugins, effects, and application packages can be done in FL Studio without opening an external browser.

  • FL Cloud: FL Cloud allows you to open and edit files on another desktop or through the mobile version of the application. Files can also be shared with other users. Users can limit specific changes while sharing files.

  • Save: Files will now automatically save to both your FL Cloud and desktop. This feature can be disabled at any time.

Improvements

  • Trial Users: Trial plugins no longer have a time limit.

Bug Fixes

  • Slicex: ‘Undo’ feature has been fixed to apply to more than one edit.

  • Fruity Reverb 2: Missing ‘Save Preset As.’

  • Keyboard shortcuts for copy/paste don’t work on the piano roll.

  • Crash when slicing a note in the piano roll.

  • Fixed crashes when opening a new project.

  • Resolved issue with easily selecting chords in the chord progression tool.

  • Maximus: Crash when the sample rate or buffer size is too high.
